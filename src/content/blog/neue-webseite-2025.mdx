## Was bietet unsere neue Webseite?

### Informationen über das Repair Café
Besucher, die noch nie bei uns waren, finden auf der Seite eine ausführliche Erklärung dessen, was ein Repair Café ist. Wir beschreiben unser Konzept und erläutern, warum Reparieren mehr bedeutet als nur Abfall zu vermeiden.

### Termine und Veranstaltungen
Alle kommenden Termine unserer Repair Café Veranstaltungen sind übersichtlich aufgelistet.

### Vorbereitung für Besucher
Für alle, die Gegenstände zur Reparatur mitbringen möchten, haben wir eine Checkliste erstellt:
- Was mitgebracht werden sollte
- Wie Geräte vorzubereiten sind
- Welche Reparaturen möglich sind und welche nicht
- Hinweise für einen erfolgreichen Besuch

### Kontaktmöglichkeiten
Die Kontaktseite listet verschiedene Wege auf, wie Interessierte mit uns in Verbindung treten können.

### Blog
Ein wichtiger Bestandteil der neuen Seite ist unser Blog mit aktuellen Berichten:
- Berichte von Veranstaltungen
- Reparatur-Tipps und Anleitungen
- Neuigkeiten rund um das Repair Café
- Beiträge von Helfern
- Pressemitteilungen und Medienberichte

## Datenschutz und Transparenz

### Keine Cookies
Unsere Webseite verwendet keine Cookies. Wir sammeln keine Nutzerdaten, verwenden kein Tracking und schalten keine Werbung. Besucher können die Seite nutzen, ohne sich Gedanken über ihre Privatsphäre machen zu müssen.

### Keine Datensammlung
Was wir nicht verwenden:
- Tracking-Scripts
- Analytics-Tools
- Social Media Pixels
- Versteckte Datensammlungen
- Weitergabe von Nutzerdaten

Dadurch bleibt die Privatsphäre der Nutzer geschützt.

### Open Source
Der komplette Code der Webseite ist open source und auf GitHub verfügbar. Jeder kann nachvollziehen, wie die Seite funktioniert.

Vorteile von Open Source:
- Transparenz durch einsehbaren Code
- Vertrauen durch keine versteckten Funktionen
- Anderen können vom Code lernen
- Langfristige Verfügbarkeit

## Technische Umsetzung

### Performance und Effizienz
Die Seite wurde mit moderner Web-Technologie entwickelt:
- Astro als Framework für Performance
- Tailwind CSS für responsives Design
- Optimierte Bilder für schnelle Ladezeiten
- Schlanker Code für bessere Performance

### Responsive Design
Die Seite passt sich automatisch an verschiedene Bildschirmgrößen an - ob Desktop, Tablet oder Smartphone.

### Barrierefreiheit
Bei der Entwicklung wurde Wert auf Barrierefreiheit gelegt:
- Optimierung für Screenreader
- Ausreichende Kontraste
- Klare Navigation
- Alternative Texte für Bilder

## Die neue Webseite besuchen

Die neue Webseite ist unter www.repaircafe-xanten.de erreichbar. Feedback, Fehlermeldungen und Verbesserungsvorschläge nehmen wir gerne entgegen - per E-Mail oder beim nächsten Repair Café.
